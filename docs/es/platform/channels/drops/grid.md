# Grilla

Contiene la definición una Grilla en una Página. 

## grid.id

El id del grid.

### output

```508```

## grid.cache_key

La key del cache asociada al grid.

### output

```layouts/grids/full_grids/508```

## grid.resolve_type

El tipo del grid con underscore.

### output

```full_grid```
